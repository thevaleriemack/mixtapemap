About

This website is a capstone project for INFO/CS 5100 at Cornell University. Users can create a 'mixtape' (aka playlist) by selecting songs that they like. Data was collected by way of AJAX calls to the Spotify API which makes for a dynamic and interactive experience for our users.

Begin by typing the name of a music artist and clicking start or pressing the return key. A popular song by that artist will be randomly chosen as the first song in the mixtape. Five new song options from similar artists will appear for every new song that gets added to the playlist. The user will be able to see a data visualization of how different songs relate to each other and compare the results to their own expectations and preferences.
